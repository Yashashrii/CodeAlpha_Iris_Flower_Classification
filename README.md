# Iris Flower Classification using Machine Learning

## Project Overview

This project predicts the species of an Iris flower using Machine Learning.

The model is trained using the famous Iris dataset and the Random Forest Classification algorithm.

---

## Dataset

The dataset contains 150 flower samples.

Features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target:

- Iris-setosa
- Iris-versicolor
- Iris-virginica

---

## Libraries Used

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

---

## Machine Learning Steps

- Import libraries
- Load dataset
- Data exploration
- Data preprocessing
- Split training and testing data
- Train Random Forest Classifier
- Predict flower species
- Evaluate accuracy
- Visualize results
